Falco Artifact Operator container
=================================

This image builds the artifact operator (`./cmd/artifact`) of [falco-operator](https://github.com/falcosecurity/falco-operator), which manages Falco artifacts such as rules and plugins.

Docker images
-------------

Docker images are available on [ghcr.io](https://github.com/cybozu/neco-containers/pkgs/container/falco-artifact-operator)
