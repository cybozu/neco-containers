Falco Operator container
========================

This image builds the instance operator (`./cmd/instance`) of [falco-operator](https://github.com/falcosecurity/falco-operator), which deploys and manages Falco instances on Kubernetes.

Docker images
-------------

Docker images are available on [ghcr.io](https://github.com/cybozu/neco-containers/pkgs/container/falco-operator)
