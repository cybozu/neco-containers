falco-k8s-metacollector container
===========================

[falco-k8s-metacollector](https://github.com/falcosecurity/k8s-metacollector) fetches metadata from the Kubernetes API server and dispatches them to Falco instances.

Docker images
-------------

Docker images are available on [ghcr.io](https://github.com/cybozu/neco-containers/pkgs/container/falco-k8s-metacollector)
