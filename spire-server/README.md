spire-server
============

This directory provides a Dockerfile to build a Docker container for [SPIRE Server](https://github.com/spiffe/spire).

SPIRE Server is the server component of SPIRE (the SPIFFE Runtime Environment).

Docker images
-------------

Docker images are available on [ghcr.io](https://github.com/cybozu/neco-containers/pkgs/container/spire-server)
