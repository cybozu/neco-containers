spiffe-csi-driver
=================

This directory provides a Dockerfile to build a Docker container for [SPIFFE CSI Driver](https://github.com/spiffe/spiffe-csi).

SPIFFE CSI Driver is a Container Storage Interface driver for Kubernetes that facilitates injection of the SPIFFE Workload API.

Docker images
-------------

Docker images are available on [ghcr.io](https://github.com/cybozu/neco-containers/pkgs/container/spiffe-csi-driver)
