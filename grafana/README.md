Grafana container
==================

[Grafana](https://grafana.com/) is the dashboard for Prometheus.

Usage
-----

1. Run the container
    ```console
    $ docker run --net=host quay.io/cybozu/grafana:5.4.3-1
    ```
2. Access the browser `localhost:3000`
3. Input user/password, the default values are admin/admin
