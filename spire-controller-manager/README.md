spire-controller-manager
========================

This directory provides a Dockerfile to build a Docker container for [SPIRE Controller Manager](https://github.com/spiffe/spire-controller-manager).

SPIRE Controller Manager is a Kubernetes controller that facilitates SPIFFE identity management for Kubernetes workloads.

Docker images
-------------

Docker images are available on [ghcr.io](https://github.com/cybozu/neco-containers/pkgs/container/spire-controller-manager)
